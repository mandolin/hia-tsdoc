# Fixtures

`doc-source-map/ts-js` contains the first generated-source TSDoc fixture:

- `src/calculator.ts` is the TypeScript source.
- `dist/calculator.js` is generated by TypeScript.
- `dist/calculator.js.map` is the ordinary source map.
- `dist/calculator.tsdoc.json` is the TSDoc extraction artifact.
- `dist/calculator.jsdoc-bridge.json` is the TS-to-JSDoc bridge artifact.
- `dist/calculator.docmap.json` is the documentation source-map manifest.
- `dist/calculator.hia.json` is the HIA core document fixture.

Generate and check artifacts with:

```sh
npm run build:fixtures
npm run check:fixtures
```
